## [1.0.0] - 2026-06-28

### ✨ Initial Release

The complete, production-ready version of the uv Beginners Handbook.

#### Added

**Core Content (14 Parts)**
- Part 1: The Foundations (What is Python, packages, uv's purpose)
- Part 2: Getting Started (Installation, first commands)
- Part 3: Key Concepts (Virtual environments, dependencies, lock files)
- Part 4: Creating & Managing Projects (Init, structure, adding/removing deps)
- Part 5: Working With Python Versions (Installing, managing, pinning)
- Part 6: Running Code (Projects, scripts, script dependencies)
- Part 7: Tools & Utilities (Running tools, installing, managing)
- Part 8: The pip Interface (Traditional workflows)
- Part 9: Advanced Topics (Workspaces, publishing, caching, configuration)
- Part 10: Decision Trees & Workflows (Flowcharts, complete workflows)
- Part 11: Command Reference (Complete reference for all commands)
- Part 12: Cheat Sheets (Essential, daily workflow, troubleshooting)
- Part 13: Comparison Tables (uv vs other tools)
- Part 14: Real-World Examples (4 complete projects)

**Additional Content**
- Appendix: Troubleshooting & FAQ (Common problems and answers)
- Glossary of Terms (Definitions for all key concepts)
- Comprehensive Table of Contents
- Internal navigation links throughout

**Repository Files**
- README.md (Welcome and navigation)
- CONTRIBUTING.md (Contribution guidelines)
- CODE_OF_CONDUCT.md (Community standards)
- CHANGELOG.md (This file)
- SECURITY.md (Security guidelines)
- LICENSE (MIT License)
- .gitignore (Git configuration)

**Examples Directory** (`examples/`)
- 01-hello-world: Basic "Hello World" project
- 02-web-api: REST API with FastAPI
- 03-data-analysis: Data analysis with pandas and Jupyter
- 04-ai-chatbot: OpenAI chatbot with Flask

**Features**
- ✅ Beginner-friendly language (no assumed knowledge)
- ✅ Real-world examples (copy-paste ready)
- ✅ Decision trees (visual flowcharts)
- ✅ Cheat sheets (quick reference)
- ✅ Complete command reference
- ✅ Troubleshooting guide
- ✅ Professional formatting
- ✅ Internal navigation links
- ✅ Code highlighting
- ✅ Table formatting
